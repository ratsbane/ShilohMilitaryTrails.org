Unit Application Forms


The unit application has three parts.
Page 1 give basic information about all trails and cost information on patches and pins.   Unit Application Page 1
Page 2 is the form used to order reading materials for each hike. (Before You Hike)     Pre Hike Order Form
Page 3 is used to order patches, pins, and Shiloh Veteran Hiker Certificate  when you return your credentials cards for the trails hiked.   Awards Order Form.

 

Instructions for hiking the trails, credential cards, and the maps needed can be printed from the web pages.  If you find errors in instructions please let us know so we can improve our documentation.
